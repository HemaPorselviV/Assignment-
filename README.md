Cybersecurity is the practice of protecting systems, networks, programs, and data from digital attacks, unauthorized access, or damage. It ensures the confidentiality, integrity, and availability of information using tools, processes, and behaviors to mitigate risks from malware, social engineering, and system vulnerabilities. 

Key Components of Cybersecurity:
Network Security: Securing networks from intruders using firewalls and VPNs.
Application Security: Protecting software and devices via secure coding and vulnerability assessments.
Information/Data Security: Safeguarding data integrity and privacy, both in storage and transit, often using encryption.
Cloud Security: Protecting data and infrastructure within cloud computing environments.
Operational Security: Managing processes for handling, protecting, and accessing data assets. 

Common Cyber Threats:
Malware: Malicious software including viruses, worms, and spyware.
Ransomware: Encrypting data to demand payment.
Phishing: Using fraudulent emails/messages to steal sensitive information.
Social Engineering: Manipulating individuals into revealing confidential information.
SQL Injection: Injecting malicious code to manipulate databases. 

Key Principles and Defenses:
CIA Triad: The foundation of cybersecurity focusing on Confidentiality, Integrity, and Availability.
Defense in Depth: Using multiple, overlapping layers of protection.
Authentication/Authorization: Ensuring only verified users access specific resources.
User Education: Teaching safe internet habits to prevent accidental breaches.
